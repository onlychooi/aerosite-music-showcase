# Aero Music

[View the live site](https://onlychooi.github.io/aerosite-music-showcase/)

A glossy futuristic music portal built with HTML, CSS, and JavaScript.  
This project blends Frutiger Aero atmosphere, glassmorphism panels, local audio playback, and looping visual mood videos into a nostalgic-digital showcase.

## Features

- Local audio playback
- Looping video mood panels
- Glossy portal-style interface
- Frutiger Aero / Y2K-inspired visual design
- Responsive one-page layout
- Real external link support

## Built With

- HTML
- CSS
- JavaScript

## Media Setup

The project uses local media files stored in:

- `audio/`
- `video/`

Example files used in this project:

- `audio/track-01.wav`
- `audio/track-02.wav`
- `audio/track-03.wav`
- `audio/track-04.wav`
- `video/cloud-glass.mp4`
- `video/chrome-light.mp4`
- `video/rain-signal.mp4`
- `video/night-drive.mp4`

## Notes

Some media files are large, so the live site may load more slowly than a typical static site.  
A future improvement would be compressing the audio and video files for better performance.
